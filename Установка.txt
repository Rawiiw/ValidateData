Перед запуском обязательно скачайте python 3.11 либо из файла в папке под названием python-3.11.0-amd64.exe, либо с официального сайта 
https://www.python.org/downloads/release/python-3110/

После скачивания и установки python 

Запуск:

1. Скачайте папку ValidateData;
2. Запустите CMD консоль от имени администратора;
3. Перейдите в папку с проектом с помощью команды cd (например, "cd C:\Users\Alexandra\Documents\ValidateData" (укажите свой полный путь до папки));
4. Введите команду "pip install -r requirements.txt";
5. После установки введите команду "python main.py"

Появится окно Проводника выбора файла с наземными данными

 Если появляется ошибка, что библиотеки не установлены какие-то, нужно написать команду "pip install [название библиотеки из ошибки]"
